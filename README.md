# AJAX CLICK V7.1

Next.js functional starter for the AJAX CLICK AI Creative Workspace.

## Run

```bash
npm install
npm run dev
```

Open `http://localhost:3000`.

## Added in V7.1

- Real reference image previews
- Remove and replace references
- Local project save
- Projects dashboard
- Open and delete saved projects
- Local version history
- Restore prior versions
- Generated gallery
- Local browser persistence with `localStorage`

## Important

This version stores project data and image previews in the browser. Large image files may eventually exceed browser storage limits. Production storage should use a database and object storage.
