User Guide

1 ကြိုဆိုပါတယ် / Welcome
ဒါကတော့ Ajax Click Photo Studio LAB Pro ဖြစ်ပါတယ်။ ဒီ Tool က ခင်ဗျားရဲ့ ဓာတ်ပုံအကြိုက်တွေကို AI နားလည်တဲ့ Professional DSLR Quality Prompt အဖြစ် ပြောင်းပေးပြီး၊ ခင်ဗျား ကြိုက်နှစ်သက်ရာ AI Model (Black Forest, Draw Things, OpenAI) တွေနဲ့ ဓာတ်ပုံထုတ်ယူနိုင်အောင် ကူညီပေးပါတယ်။
This is Ajax Click Photo Studio LAB Pro. It converts your photo preferences into a Professional DSLR Quality Prompt that AI understands, and helps you generate images using your preferred AI models (Black Forest, Draw Things, OpenAI).
2 မစတင်ခင် ပြင်ဆင်ခြင်း / Setup
အောက်ပါ API Key တွေကို ကြိုပြင်ထားဖို့ လိုပါတယ် (သုံးချင်တဲ့ဟာကိုပဲ ပြင်ထားရုံနဲ့ ရပါတယ်)။
• Gemini API – Face Analysis အတွက် အဓိက လိုအပ်ပါတယ်။ (Google AI Studio မှ အခမဲ့ရယူပါ)
• OpenAI API – Black Forest 3.0 (DALL-E) သုံးဖို့ လိုအပ်ပါတယ်။ (Paid Service)
• Draw Things – Local App နဲ့ ချိတ်ဆက်သုံးဖို့ပါ။ (Port 7861)
Set up the following API Keys (only set up the ones you want to use):
• Gemini API – Mainly required for Face Analysis. (Get it for free from Google AI Studio)
• OpenAI API – Required for Black Forest 3.0 (DALL-E). (Paid Service)
• Draw Things – Required to connect with the local Draw Things app. (Port 7861)
3 မျက်နှာပုံတင်ခြင်း & ခွဲခြမ်းစိတ်ဖြာခြင်း / Face Upload & Analysis
ပုံတင်နည်း: Left Panel က "Face References" ထဲက Man/Woman/Kid အကွက်ကို နှိပ်ပြီး ပုံရွေးပါ (သို့) Drag & Drop လုပ်ပါ။
ခွဲခြမ်းစိတ်ဖြာနည်း: ပုံတင်ပြီးရင် "Analyze" ကိုနှိပ်ပါ။ AI က မျက်နှာအသွင်အပြင်ကို ဖော်ပြပြီး Prompt ထဲမှာ အလိုအလျောက် ထည့်ပေးပါမယ်။ (Gemini API လိုအပ်)
How to Upload: Click on Man/Woman/Kid boxes in "Face References" (Left Panel) and select a photo, or Drag & Drop.
How to Analyze: Click "Analyze" after uploading. AI will describe facial features and automatically inject it into your Prompt. (Requires Gemini API)
4 ဓာတ်ပုံဆက်တင်များ / Photo Settings
Right Panel မှာ အောက်ပါအချက်တွေကို ရွေးချယ်ပါ။
• Subject – Man, Woman, Couple, Family
• Shot Type – Close-up, Medium, Full Body, Wide
• Aspect Ratio – 1:1, 16:9, 9:16
• Style / Lighting / Expression – ခင်ဗျား ကြိုက်နှစ်သက်ရာ ပုံစံ
On the Right Panel, select your preferences:
• Subject – Man, Woman, Couple, Family
• Shot Type – Close-up, Medium, Full Body, Wide
• Aspect Ratio – 1:1, 16:9, 9:16
• Style / Lighting / Expression – Choose your desired mood
5 ကိုယ်ဟန် & ဝတ်စုံ / Poses & Outfits
Pose: Man Solo, Woman Solo, Couple Pose အုပ်စုတွေကနေ ကြိုက်ရာကိုယ်ဟန်ကို နှိပ်ရွေးပါ (ရွေးချယ်လိုက်ရင် ရွှေရောင်ပြောင်းသွားမယ်)။
Outfit: Business, Evening, Casual စသည်ဖြင့် ရွေးပါ။
Ethnic: Myanmar, Kimono, Sari, Ao Dai စသည့် တိုင်းရင်းသားဝတ်စုံများ ရွေးနိုင်ပါတယ်။
Occasion: Wedding, Birthday, Professional စသည့် နောက်ခံမြင်ကွင်းကို ရွေးပါ။
Pose: Click your favorite pose from Man Solo, Woman Solo, or Couple Pose groups (selected will turn gold).
Outfit: Choose from Business, Evening, Casual, etc.
Ethnic: Select traditional attire like Myanmar, Kimono, Sari, Ao Dai.
Occasion: Pick a scene like Wedding, Birthday, Professional.
6 Prompt ထုတ်လုပ်ခြင်း & ကူးယူခြင်း / Generating & Copying Prompt
ထုတ်လုပ်နည်း: "Generate Prompt" ခလုတ်ကိုနှိပ်ပါ။ ရွေးချယ်ထားတဲ့ အချက်အားလုံးကို ပေါင်းစပ်ပြီး DSLR Quality Prompt ကို အောက်ခြေ Output Box မှာ ပြသပေးပါမယ်။
ကူးယူနည်း: "Copy" ခလုတ် သို့မဟုတ် Ctrl + C ကိုနှိပ်ပါ။
Generate: Click the "Generate Prompt" button. It combines all your selections and displays a DSLR Quality Prompt in the output box below.
Copy: Click the "Copy" button or press Ctrl + C.
7 ပုံထုတ်လုပ်ခြင်း / Image Generation
Black Forest AI – Credit (Pre-paid) ဝယ်ထားရန် လိုအပ်ပါတယ်။ Header က "Buy Credits" မှ ဝယ်ယူနိုင်ပါတယ်။
Draw Things – Local Server နဲ့ ချိတ်ဆက်ပြီး ပုံထုတ်ပေးပါတယ်။
ChatGPT App (Plus/Pro) – ChatGPT Plus/Pro အကောင့်ရှိရင် App ထဲမှာ တိုက်ရိုက်ပုံထုတ်လို့ရပါတယ် (OpenAI API Key မလိုပါ)။ ၃ နာရီလျှင် ပုံ ၄၀-၅၀ ခန့် ထုတ်လို့ရပါတယ်။
OpenAI API – ကိုယ်ပိုင် Tool ကနေ API ကိုတိုက်ရိုက်သုံးချင်ရင် API Key လိုအပ်ပြီး ငွေကုန်ကျပါမယ်။
Gemini – Prompt ကိုကူးပြီး Gemini App မှာ ကိုယ်တိုင် Generate လုပ်ပါ။
Black Forest AI – Requires Pre-paid Credits. Purchase via "Buy Credits" in the Header.
Draw Things – Connects to your local Draw Things Server.
ChatGPT App (Plus/Pro) – If you have ChatGPT Plus/Pro, generate images directly inside the App (no OpenAI API Key needed). Limit: ~40-50 images per 3 hours.
OpenAI API – Requires an API Key and is pay-per-use.
Gemini – Copy the prompt and generate manually in the Gemini App.
8 ကြိုတင်ကြည့်ရှုခြင်း & ဒေါင်းလုဒ် / Preview & Download
ဒီ Box က ဘာလုပ်ပေးတာလဲ? ထုတ်လိုက်တဲ့ပုံကို ပြသပေးတယ်၊ ပြင်ပပုံတွေကိုလည်း ပြန်ပြလို့ရတယ်။
ပုံတင်နည်း ၃ မျိုး:
၁။ Click to Upload – Box ကိုနှိပ်ပြီး ပုံရွေးပါ။
၂။ Drag & Drop – ပုံကို Box ထဲကို ဆွဲချပါ။
၃။ Paste URL – Image URL (e.g., .jpg/.png) ကိုကူးထည့်ပြီး "Load" နှိပ်ပါ။
blob: URL ဆိုရင် "Save image as" နဲ့ သိမ်းပြီး Drag & Drop လုပ်ပါ။
ဒေါင်းလုဒ်ဆွဲနည်း: "Download" ခလုတ်ကိုနှိပ်ပြီး ပုံကို သိမ်းဆည်းပါ။
What does this Box do? Displays generated images and allows you to upload external images.
3 Ways to load images:
1. Click to Upload – Click the box and select an image.
2. Drag & Drop – Drag and drop an image file directly.
3. Paste URL – Paste a direct image URL (e.g., .jpg/.png) and click "Load".
For blob: URLs – Right-click the image -> "Save image as", then Drag & Drop it here.
Download: Click the "Download" button to save the image to your device.

→ Gemini API Key ထည့်ထားလား စစ်ပါ။ (Google AI Studio မှ အခမဲ့ရယူပါ)
Draw Things မချိတ်ဘူး

→ "Save image as" နဲ့ သိမ်းပြီး Drag & Drop လုပ်ပါ။

ChatGPT API မရဘူး
→ ChatGPT App (Plus/Pro) သုံးရင် API Key မလိုပါ။ API သုံးရင်တော့ Key လိုအပ်ပါတယ်။
